# Hapi Route Status

Adds a monitor endpoint /status

## Test & Development

Running tests

	$ npm test

Watching for changes and run all tests when developing.

	$ npm run watch

Creating releases

	$ npm run release
	$ npm run release:minor
	$ npm run release:major
